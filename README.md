Pebble is a fantasy game console built around the W65C02 CPU.

Specifications:

- W65C02 @ 1 MHz
- 8 KB RAM
- 32 KB Cartridge ROM
- 64×64 Display
- 4 Grayscale Colors
- 30 FPS
- Python-based SDK

The repository contains:

- Emulator
- Compiler
- Runtime
- Studio
